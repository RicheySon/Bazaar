## Troubleshooting

If you continue getting "Address is invalid" errors:

1. **Try the bitcoincash: format first** (Format 1 above)
2. **Clear browser cache and refresh** the faucet page
3. **Disable VPN/proxy** if you're using one (faucets sometimes block them)
4. **Try a different browser** if captcha isn't working
5. **Wait a few minutes** - the faucet may have rate limiting per IP
